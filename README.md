## Installation

Install with yarn

```bash
  yarn
  yarn docker:up
  yarn init-db
  yarn start
```
    
## Run Locally

Clone the project

```bash
  git clone https://github.com/prastowo18/kitameraki-be-test.git
```

Go to the project directory

```bash
  cd kitameraki-be-test
```

Install dependencies

```bash
  yarn
```

Compose Docker

```bash
  yarn docker:up
```

Init DB

```bash
  yarn init-db
```

Start the server

```bash
  yarn start
```
